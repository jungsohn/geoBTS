Design References
------------------

**Terzaghi (1943)**  
- Key assumptions: Strip footing with homogeneous soil  
- Limitation: No shape or depth adjustment

**Meyerhof (1951,1963)**  
- Key assumptions:  
  - Apply K factors to Terzaghi's  
  - Updated Ng: For example, φ = 30 → Ng (Meyerhof) = 20, Ng (Terzaghi) = 15
